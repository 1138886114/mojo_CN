# Mojo CLI

## mojo format (格式化)

格式化 Mojo 源文件。

## 概要

```
mojo format [options] <sources...>
```

## 描述

使用 Mojo 特定的 lint 工具格式化给定的 Mojo 源集。

## 选项

### 格式选项

#### `--line-length <INTEGER>`,`-l <INTEGER>`

设置最大字符行长度。默认值为 80。

### 诊断选项

#### `--quiet`,`-q`

禁用非错误消息。

### 常用选项

#### `--help`,`-h`

显示帮助信息。
